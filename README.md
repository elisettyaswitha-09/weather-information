# 🌤 Weather Information Web App

## Overview
The Weather Information Web App is a simple and responsive web application that provides real-time weather updates for any city. It is developed using HTML, CSS, and JavaScript and uses the OpenWeatherMap API to fetch live weather information.

## Features
- Search weather by city name
- Display current temperature
- Show weather condition
- Display humidity
- Show wind speed
- Dynamic weather icons
- Responsive and modern user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## Project Structure

WeatherApp/
│── index.html
└── README.md

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Replace `YOUR_API_KEY` in the JavaScript code with your OpenWeatherMap API key.
4. Save the file.
5. Open `index.html` in any web browser.

## API
This project uses the OpenWeatherMap API to retrieve live weather information.

Website:
https://openweathermap.org/
